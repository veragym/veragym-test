앱 설치 방법
📱
VERA GYM은 별도 앱스토어 설치 없이 홈 화면에 추가하는 방식입니다.
설치 후에는 일반 앱과 동일하게 아이콘으로 실행됩니다.
🍎 iPhone (iOS) — Safari에서만 설치 가능
  1. 1
    Safari 앱을 실행합니다 Chrome, 네이버 앱 등 다른 브라우저는 설치 불가
  2. 2
    관리자에게 받은 트레이너 앱 링크를 주소창에 입력합니다
  3. 3
    하단 공유 버튼(□↑)을 누릅니다
  4. 4
    「홈 화면에 추가」를 선택합니다 이름 확인 후 「추가」 버튼 터치
  5. 5
    홈 화면의 VERA GYM 아이콘으로 실행합니다
🤖 Android — Chrome에서 설치
  1. 1
    Chrome 브라우저로 트레이너 앱 링크에 접속합니다
  2. 2
    주소창 우측 또는 하단에 「앱 설치」 팝업이 나타납니다 팝업이 없으면 Chrome 메뉴(⋮) → 「홈 화면에 추가」
  3. 3
    「설치」를 눌러 홈 화면에 추가합니다
🔄
업데이트는 자동으로 적용됩니다
앱을 완전히 종료 후 다시 실행하면 최신 버전이 자동으로 반영됩니다. 재설치나 캐시 삭제는 필요 없습니다.
2
로그인
🔐 로그인 방법
  1. 1
    관리자에게 발급받은 이메일과 비밀번호를 입력합니다
  2. 2
    로그인 버튼을 누릅니다
  3. 3
    헤더에 이름·지점명이 표시되면 로그인 완료 예: 김트레이너 · 강남점
💡
로그아웃
화면 우측 상단 로그아웃 버튼을 누르면 됩니다. 본인의 기기가 아닐 때 반드시 로그아웃하세요.

4
내 회원 관리
➕ 회원 추가
  1. 1
    「내 회원」 탭에서 + 회원 추가 버튼을 누릅니다
  2. 2
    회원 이름을 입력합니다 메모란에 특이사항, 목표 등 추가 정보를 입력할 수 있습니다 (선택)
  3. 3
    추가 버튼을 눌러 저장합니다
🔗 회원 앱 링크 전달

회원이 본인의 수업 기록과 사진을 볼 수 있는 전용 링크를 제공할 수 있습니다.

  1. 1
    회원 이름을 눌러 상세 화면으로 이동합니다
  2. 2
    상단에 표시된 링크 박스를 누르면 클립보드에 복사됩니다
  3. 3
    카카오톡 등으로 회원에게 링크를 전달합니다 회원은 이 링크로 수업 기록, 운동 사진을 확인할 수 있습니다
📋 회원별 수업 기록 확인

회원 상세 화면 하단에서 해당 회원의 전체 수업 기록을 확인할 수 있습니다.
각 기록에서 수정 삭제 버튼으로 편집 또는 삭제할 수 있습니다.

💡
회원 목록으로 돌아가기
회원 상세에서 좌측 상단 목록 버튼을 누르면 회원 목록으로 돌아옵니다.
5
수업일지 작성
📍
수업일지 작성 진입 방법
내 회원 탭 → 회원 선택 → 하단 ＋ 작성 버튼
📅 날짜 선택

상단 날짜 칩을 누르면 날짜 선택 창이 열립니다. 기본값은 오늘 날짜입니다.

🏋️ 운동 추가
  1. 1
    하단 ＋ 운동 추가 버튼을 누릅니다
  2. 2
    「운동 라이브러리」 모달이 열립니다 상단 검색창에서 운동명을 검색하거나, 부위·도구 필터를 사용해 운동을 찾습니다
  3. 3
    원하는 운동을 눌러 추가합니다 자주 사용한 운동은 상단 「자주 사용」 섹션에 표시됩니다

원하는 운동이 없으면 모달 하단 라이브러리 수정 버튼으로 운동DB에서 추가할 수 있습니다.

⚖️ 세트 · 무게 · 횟수 입력
  • 추가된 운동 카드 안에 세트별로 무게(kg)와 횟수를 입력합니다
  • ＋ 세트 추가 버튼으로 세트를 늘릴 수 있습니다
  • 각 세트 우측 「—」 버튼으로 해당 세트를 삭제합니다
  • 운동 카드 하단 코멘트란에 해당 운동에 대한 메모를 남길 수 있습니다 (선택)
📸 운동 사진 촬영

시작 자세와 끝 자세 각각 1장씩 촬영할 수 있습니다.

  1. 1
    운동 카드 우측 사진 ●● 버튼을 누릅니다 ● 은 미촬영, 색이 채워진 ● 은 촬영 완료 상태입니다
  2. 2
    「시작 자세」 또는 「끝 자세」를 선택합니다
  3. 3
    「카메라로 촬영」 또는 「갤러리에서 선택」을 누릅니다
  4. 4
    사진 선택 후 자동으로 저장됩니다 저장 완료 후 다시 자세 선택 화면으로 돌아옵니다. 나머지 자세도 동일하게 촬영합니다.
  5. 5
    모두 촬영했으면 「닫기」 버튼을 눌러 사진 모달을 닫습니다
📝 전체 메모

운동 목록 하단의 메모란에 오늘 수업 전체에 대한 내용을 자유롭게 입력합니다 (선택).

운동 삭제

운동 카드 우측 상단 삭제 버튼을 누르면 확인 팝업 후 해당 운동이 제거됩니다.

🚨
반드시 「저장 후 나가기」 버튼을 눌러야 저장됩니다!
화면 우측 상단 저장 후 나가기 버튼을 눌러야만 수업 일지가 저장됩니다.

스마트폰 하드웨어 뒤로가기 버튼, 스와이프 제스처, 다른 앱 전환 등으로 나가면 작성 내용이 모두 사라집니다.
6
수업일지 수정 · 삭제
✏️ 수업일지 수정 (2가지 경로)

어느 경로든 동일한 편집 화면으로 연결됩니다.

경로 방법
수업일지 탭 수업 일지 카드 하단 수정 버튼
내 회원 탭 회원 상세 → 수업 기록 → 해당 일지 수정 버튼
🚨
수정 후에도 반드시 「저장 후 나가기」를 눌러야 저장됩니다!
수정 완료 후 저장 후 나가기 버튼을 눌러야 변경 사항이 저장됩니다.
다른 방법으로 나가면 수정 내용이 반영되지 않습니다.
🗑️ 수업일지 삭제

수업 일지 카드 하단 삭제 버튼을 누르면 확인 팝업이 뜨고, 확인 시 삭제됩니다.
※ 삭제된 일지는 관리자가 복구할 수 있습니다.

7
운동DB
💡
운동DB 진입 방법 3가지
① 상단 탭 「운동DB」 직접 진입
② 수업일지 작성 중 「라이브러리 수정」 버튼
③ 운동 추가 모달에서 라이브러리가 비어있을 때 나타나는 버튼
📂 전체 운동 탭

관리자가 등록한 모든 운동의 목록입니다.

  • 검색: 상단 검색창에 운동 이름을 입력해 검색합니다
  • 필터: 부위(전신, 상체, 하체 등) 또는 도구로 필터링할 수 있습니다
  • ＋ 추가 버튼을 눌러 내 라이브러리에 운동을 등록합니다
  • 등록된 운동은 추가됨 / 해제 버튼으로 표시되며, 누르면 확인 팝업 후 제거됩니다
  • 운동 이름을 누르면 설명·이미지 상세 보기가 열립니다
⭐ 운동 라이브러리 탭

내가 추가한 운동만 보이는 내 전용 목록입니다. 수업일지 작성 시 이 목록에서 운동을 선택합니다.

  • 즐겨찾기 ★: 자주 사용하는 운동에 별표를 눌러 즐겨찾기로 등록합니다. 즐겨찾기 운동은 목록 최상단에 표시됩니다
  • 사용 횟수: 수업일지 작성 시 사용된 횟수가 자동 기록되어 표시됩니다
  • 추가됨 / 해제 버튼으로 라이브러리에서 제거할 수 있습니다
⚠️
수업일지 작성 중 라이브러리 수정 후 돌아오기
수업일지 작성 화면에서 「라이브러리 수정」 버튼으로 운동DB에 진입했을 경우,
운동 추가 완료 후 반드시 상단의 ← 수업으로 돌아가기 버튼을 눌러야 합니다.

스마트폰 뒤로가기를 사용하면 작성 중이던 수업일지 내용이 사라질 수 있습니다.
8
요청사항
📬 요청사항 작성

관리자에게 전달할 내용을 요청사항 탭에서 작성합니다.

유형 설명
운동DB 추가 요청 라이브러리에 없는 운동 추가를 요청할 때
버그/오류 신고 앱에서 오류가 발생했을 때 신고
기타 그 외 전달사항

요청사항 목록에서 처리 상태(처리중 / 완료)를 확인할 수 있습니다.

9
⚠️ 필수 주의사항
🚫
스마트폰 뒤로가기 버튼 사용 금지
수업일지 작성·수정 화면에서 안드로이드 뒤로가기 버튼, iOS 스와이프 제스처를 사용하면 작성 중인 내용이 모두 사라집니다.

반드시 앱 화면 안의 버튼으로만 이동하세요.
💾
「저장 후 나가기」 버튼 필수
수업일지 작성·수정 후 저장 후 나가기 버튼을 눌러야만 저장됩니다.

앱을 강제 종료하거나 다른 앱으로 이동하면 저장되지 않습니다.
↩️
운동DB에서 수업일지로 돌아올 때
수업일지 작성 중 운동DB로 이동했다면, 반드시 ← 수업으로 돌아가기 버튼을 눌러야 기존 작성 내용이 유지됩니다.
📸
사진 촬영 후 「닫기」 버튼 필수
시작·끝 자세 사진을 모두 촬영한 후 닫기 버튼을 눌러 사진 모달을 닫아야 합니다. 닫기 전에도 사진은 저장되어 있습니다.
✅ 올바른 수업일지 작성 흐름 요약
  1. 내 회원 탭 → 회원 선택 → 하단 ＋ 작성 버튼
  2. 날짜 확인 및 수정
  3. 운동 추가 → 검색 또는 필터로 운동 선택
  4. 세트별 무게·횟수 입력
  5. 필요 시 사진 촬영 → 닫기
  6. 필요 시 코멘트, 전체 메모 입력
  7. 저장 후 나가기 버튼 클릭 ← 가장 중요!
VERA GYM 트레이너 앱 사용 안내서
문의 및 오류 신고는 앱 내 「요청사항」 탭을 이용해 주세요.
